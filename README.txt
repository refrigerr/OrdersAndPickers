This program solves exercises for intership in Ocado. First exercise is about maximizing number of orders.
Second exercise is about maximizing value of orders.

Dictionary contains zipped project and 2 .jar files.
zadanie1.jar is for first exercise and zadanie2.jar is for second exercise.
To run any of these files from terminal you have to type:

java -jar absolute/path/to/jar/file.jar absolute/path/to/store/configuration.json absolute/path/to/list/of/orders.json
